This is external template inside `` component.

URL like templates are fetched from the server and rendered inside the target element.

It is shown using directive x-render and cached so subsequent rendering is done in browser: 

            <div x-render.cache="'ext.html?$target=#tabext'"><div>

Toggle ext.html inside the div below on click using directive x-template 

            <div x-template="ext ? 'ext.html' : ''">
